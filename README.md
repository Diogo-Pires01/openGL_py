# Projeto OpenGL em Python
---

## Requisitos
Antes de rodar o projeto, instale o Python 3.10 (ou superior)

Instale as seguintes libs:
```
pip install PyOpenGL PyOpenGL_accelerate glfw Pillow numpy PyGLM
```

## Para executar:
```
python main.py
```
